Simple yet powerful WordPress plugin to get feedback and show with powerful and advanced shortcodes.
 
== Description ==

Get feedback from users in easy and simple way by using the shortcode.

The feedback will store securely in the WordPress database, only admins can see the feedback with powerful Ajax pagination.

== Installation ==

1. Upload `esf-multifeed` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Easy Social Feed > Facebook > How to use?.
4. Select pages from the list and generate the shortcode.
